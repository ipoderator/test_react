# Инструкция по запуску сервера

## Проблема: Сервер не запускается

Скорее всего проблема в том, что зависимости не установлены.

## Решение:

### Вариант 1: Использовать скрипт запуска

```bash
cd /Users/glebchurkin/products-app
./start-server.sh
```

### Вариант 2: Ручная установка и запуск

Откройте терминал и выполните:

```bash
cd /Users/glebchurkin/products-app

# 1. Установите зависимости (это может занять несколько минут)
npm install

# 2. После успешной установки запустите сервер
npm run dev:5002
```

### Вариант 3: Если npm install не работает

Попробуйте:
```bash
cd /Users/glebchurkin/products-app
rm -rf node_modules package-lock.json
npm cache clean --force
npm install
npm run dev:5002
```

## Проверка:

После запуска вы должны увидеть:
```
▲ Next.js 14.0.4
- Local:        http://localhost:5002
```

Откройте в браузере: http://localhost:5002

## Если все еще не работает:

1. Проверьте версию Node.js: `node --version` (нужна версия 18+)
2. Проверьте версию npm: `npm --version`
3. Проверьте, не занят ли порт 5002: `lsof -i :5002`
4. Если порт занят, убейте процесс: `kill -9 $(lsof -ti:5002)`

## Ошибки компиляции:

Если видите ошибки типа "Cannot find module 'react'", это означает что зависимости не установлены. 
Выполните `npm install` еще раз.
