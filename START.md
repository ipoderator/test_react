# Инструкция по запуску сервера

Для запуска сервера на порту 5002 выполните следующие команды в терминале:

```bash
cd /Users/glebchurkin/products-app

# 1. Установите зависимости (если еще не установлены)
npm install

# 2. Запустите сервер на порту 5002
npm run dev:5002
```

Или используйте прямой вызов:

```bash
cd /Users/glebchurkin/products-app
npm install
npx next dev -p 5002
```

После успешного запуска вы увидите сообщение:
```
▲ Next.js 14.0.4
- Local:        http://localhost:5002
```

Откройте в браузере: http://localhost:5002

Если возникнут ошибки, проверьте:
1. Установлен ли Node.js: `node --version` (должна быть версия 18+)
2. Установлен ли npm: `npm --version`
3. Свободен ли порт 5002: `lsof -i :5002`
