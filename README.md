# Products App

SPA приложение для отображения списка продуктов с использованием Next.js, TypeScript и Zustand.

## Возможности

- Просмотр списка продуктов с изображениями
- Лайки продуктов (избранное)
- Удаление продуктов
- Фильтрация по всем товарам и избранным
- Детальная страница продукта
- **Создание новых продуктов** (форма с валидацией)
- **Редактирование продуктов**
- **Поиск продуктов** (в реальном времени)
- **Пагинация** списка продуктов
- **Расширенная фильтрация** (по категориям и цене)
- Адаптивный дизайн

## Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - типизация
- **Zustand** - управление состоянием
- **Tailwind CSS** - стилизация
- **Axios** - HTTP запросы
- **Fake Store API** - источник данных

## Установка

```bash

```

## Запуск

```bash

```

## Основные функции

### Страница продуктов (/products)

- Отображение всех продуктов в виде карточек
- Иконка лайка для добавления в избранное
- Иконка удаления для удаления продукта
- Фильтр для просмотра всех товаров или только избранных
- **Поиск продуктов** (без кнопки отправки, работает в реальном времени)
- **Фильтрация по категориям**
- **Фильтрация по диапазону цен**
- **Пагинация** (12 товаров на странице)
- Клик по карточке ведет на страницу детальной информации
- Кнопка "Создать продукт" для добавления нового товара

### Страница продукта (/products/:id)

- Подробная информация о продукте
- Кнопка возврата на главную страницу
- **Кнопка редактирования** для перехода на страницу редактирования

### Страница создания продукта (/create-product)

- Форма создания нового продукта
- Валидация всех полей:
  - Название (минимум 3 символа)
  - Цена (положительное число)
  - Описание (минимум 10 символов)
  - Категория (обязательное поле)
  - URL изображения (валидация URL)
- Сохранение продукта в store

### Страница редактирования продукта (/products/:id/edit)

- Форма редактирования существующего продукта
- Те же правила валидации, что и при создании
- Предзаполненные поля с текущими данными продукта
- Сохранение изменений в store

## API

Используется [Fake Store API](https://fakestoreapi.com/) для получения данных о продуктах.
